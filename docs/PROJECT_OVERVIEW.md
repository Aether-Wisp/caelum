# Caelum 项目概述

## 1. 项目简介

Caelum 是一个面向网络安全攻防教学与授权测试场景的智能化自动渗透测试平台。系统以大语言模型为决策核心，联动扫描、漏洞验证、利用与后渗透工具链，实现从资产发现到风险输出的端到端自动化流程。

平台重点支持多节点、分层网络环境下的测试任务，可在复杂靶场中通过自动代理与隧道机制完成内网纵深探测与横向验证。

## 2. 核心创新点 (Innovation Highlights)

本项目在传统的自动化探测基础上，实现了多个关键维度的架构和算法创新，尤其在“大模型深度集成”与“多层内网并发穿透”方面展现出企业级解决方案的雏形：

1. **大模型赋能的自主决策链路 (AI-Agent Pentesting)**
   Caelum 彻底颠覆了传统漏洞扫描器基于固定规则库和静态指纹匹配的模式。平台通过引入 MCP (Model Context Protocol) 标准通信协议，将大语言模型（如 GPT-4, DeepSeek 等）化作“黑客大脑”。当 70+ 底层安全工具传回原始输出日志，系统不再需要人工介入提取，而是由大模型自主理解、梳理线索，并决策下一步攻击载荷（Payload）和工具参数，实现完整的 **Human-In-The-Loop** 或者全自动渗透循环。

2. **动态无感知内网穿透技术 (Dynamic Pivoting & Tunnels)**
   在真实的攻防实战以及 Bugku PAR 等多节点高阶靶场中，获取边界跳板机权限仅仅是开始，如何将流量牵引入内网才是痛点。
   本系统创新性地实现了**动态代理链路自协商与下发**：AI 侦测到内网网段后，会指引 `session_manager` 自动从资源池挑选空闲端口，并自动化注入 `FRP` / `Chisel`，最终将 `Proxychains4` 配置映射挂载。整个过程无需任何人工敲击路由命令，直接将复杂深层网络“透明化”。

3. **高并发与异步任务调度引擎 (High-Concurrency Engine)**
   为了大幅缩短针对 C 段 / B 段庞大资产池以及全端口密集扫描的时间消耗，Caelum 的底层执行层（`tools_framework.py` 与 `server.py`）采用了极致优化的异步 I/O (Asynchronous I/O) 与协程并发架构。
   - **多目标并行处理**：支持将探活与指纹识别任务分发给集群并行处理。
   - **载荷投递并发限制与退避**：在保证不击穿目标服务器的情况下实现最大化的测试吞吐量，极大提升了网络探测与弱口令爆破场景的效能。

4. **跨平台重型武器库的容器化降维打击**
   将渗透测试人员最头疼的环境依赖问题完美解决。通过高度精简且具备系统特权的 Docker 镜像构建技术，把原本只能在 Kali Linux 平台上运行的 Nmap (依赖 Raw Socket)、Masscan、Metasploit 等重量级依赖服务，全部打包装载，让 Windows 和 macOS 开发者也能实现一键部署与原生级无损调用。

## 3. 核心能力

- **AI 驱动的任务编排**：根据实时扫描结果动态选择测试策略与工具。
- **自动化攻击链执行**：覆盖侦察、漏洞发现、利用验证、后渗透等阶段。
- **多层内网穿透支持**：原生集成 FRP、Chisel、Proxychains4 等能力。
- **高并发渗透测试调度**：高效的多线程/异步执行引擎，支持规模化并行扫描与漏扫。
- **工具生态完备**：集成 70+ 安全工具，支持按目标场景灵活调用。
- **可视化与报告化**：提供过程回显、发现汇总及结构化报告输出。

## 4. 技术架构

项目采用模块化 Python 架构，核心模块包括：

- server: Web 服务与接口入口。
- tools_framework: 安全工具统一封装与调度层。
- ai_analyzer: 大模型分析与决策适配层。
- session_manager: 会话与端口资源管理、代理链路协同。
- payload_generator: 载荷生成与利用参数构建。
- mcp: 模型上下文通信协议适配。

该架构实现了业务逻辑、模型接口与工具执行层解耦，便于扩展新工具与新模型。

## 4. 部署方式

Caelum 以 Docker Compose 作为推荐部署方式，兼顾环境一致性与跨平台可用性（Windows/Mac/Linux）。在容器化方案下，可快速完成依赖加载并对接完整工具链。

同时项目也支持本地开发模式，便于调试与二次开发。

## 5. 适用场景

- 高校网络安全课程教学与实验
- CTF/靶场训练与攻防演练
- 授权环境下的安全评估与流程验证
- 自动化测试平台原型开发与能力扩展

## 6. 项目价值

- 提升测试效率：减少重复性人工操作，缩短验证周期。
- 降低使用门槛：通过统一流程与交互界面降低学习成本。
- 强化实战能力：在多节点网络中验证真实攻防链路。
- 便于成果展示：支持流程可视化和报告化输出，适合竞赛与汇报。

## 7. 合规与边界

本项目仅用于教学、竞赛和经授权的安全测试活动。严禁用于任何未授权入侵、破坏或其他违法行为。使用者需自行确认测试目标与行为合法性，并承担相应责任。
